GITHUB VERSION CONTROL – PUSH, PULL, BRANCH & MERGE GUIDE

--------------------------------------------------

PART 1: CREATE GITHUB ACCOUNT

1. Open browser
2. Go to: https://github.com
3. Click "Sign up"
4. Enter Email, Password, Username
5. Verify and login

--------------------------------------------------

PART 2: INSTALL GIT

1. Go to: https://git-scm.com/downloads
2. Download and install Git
3. Open CMD and type:
   git --version

--------------------------------------------------

PART 3: SET UP GIT

git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

--------------------------------------------------

PART 4: CREATE REPOSITORY

1. Go to GitHub
2. Click "+" → New repository
3. Enter repo name
4. Click "Create repository"

--------------------------------------------------

PART 5: INITIAL SETUP IN LOCAL PROJECT

1. Create folder and add files
2. Open terminal in that folder

Commands:

git init
git add .
git commit -m "Initial commit"

--------------------------------------------------

PART 6: CONNECT TO GITHUB

git remote add origin https://github.com/your-username/repo-name.git
git branch -M main
git push -u origin main

--------------------------------------------------

PART 7: PUSH (UPLOAD CHANGES)

After making changes:

git add .
git commit -m "Updated code"
git push

(Push sends your code to GitHub)

--------------------------------------------------

PART 8: PULL (DOWNLOAD CHANGES)

To get latest code from GitHub:

git pull origin main

(Pull brings updated code from GitHub)

--------------------------------------------------

PART 9: CREATE BRANCH

To create new branch:

git branch feature1

To switch to branch:

git checkout feature1

OR (shortcut):

git checkout -b feature1

--------------------------------------------------

PART 10: WORK IN BRANCH

Make changes in files

Then:

git add .
git commit -m "Changes in feature branch"
git push origin feature1

--------------------------------------------------

PART 11: MERGE BRANCH

Step 1: Go back to main branch

git checkout main

Step 2: Merge branch into main

git merge feature1

--------------------------------------------------

PART 12: PUSH MERGED CODE

git push origin main

--------------------------------------------------

PART 13: DELETE BRANCH (OPTIONAL)

git branch -d feature1

--------------------------------------------------

SUMMARY

PUSH:
git push

PULL:
git pull origin main

BRANCH:
git checkout -b branch-name

MERGE:
git checkout main
git merge branch-name
